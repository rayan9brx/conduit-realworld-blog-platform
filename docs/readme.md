# Gettings Started

## Command Line Usage

- Install Java Runntime for your System.
<!-- - Donwload `plantuml-1.2024.3.jar` from the (PlantUML latest release)[https://github.com/plantuml/plantuml/releases/latest] Site. -->
- Visit the [C4 PlantUML](https://github.com/plantuml-stdlib/C4-PlantUML#getting-started) Page and read the "Getting Started" Section.
  There are also some example files in this folder.
- Run `java -jar .\plantuml-1.2024.3.jar -tsvg -output ../out diagrams/*.puml` to convert every puml in `diagrams` to a svg image in `out`.
